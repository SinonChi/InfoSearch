Для перехода на другой канал обновлений измените параметр app.update.channel на: release для релизов, beta для бета-версий, nightly для ежедневных сборок или default для отключения обновлений. Параметр меняется либо путём установки расширения Update Channel Selector, либо редактированием файла <каталог установки>\defaults\pref\channel-prefs.js Для отключения проверки совместимости дополнений в about:config создайте параметр extensions.checkCompatibility.4.0b типа boolean (логическое) и установите его значение в false Изменения Изменения в интерфейсе темы по-умолчанию.
