Установка никак не затронет текущую установку Google Chrome, вы не можете сделать Google Chrome Canary Build вашим браузером по умолчанию, доступен только для Windows. Живет своей жизнью: свой каталог для установки, отдельно обновляется, может быть запущен параллельно с stable/beta/dev Google Chrome. Как говорят разработчики Canary Build тоже самое что dev, только будет чаще обновляться, версия выше чем у dev. Разработчики хотят сделать частоту обновления Canary Build практически ежедневной.
