В прошлом году я вернулся к использованию бесплатного кроссплатформенного редактора Komodo Edit для проектов на HTML, JS, PHP. Главной причиной перехода с Aptana IDE явилась более высокая скорость работы и неожиданный отказ Aptana от поддержки собственного расширения для PHP в пользу PDT. Хотелось бы рассказать об изменениях, которые нас ждут в 6-ой версии на примере вышедшей 14 июля Beta 2: Обновление кода до Mozilla 1.9.1 (используется в firefox 3.5), редактор работает намного быстрее; Поддержка HTML5, CSS3 (в том числе автодополнение специфичных для различных браузеров свойств: -moz-, -webkit-); Новый файловый менеджер Places (пользуйтесь осторожнее, встречал коварные баги, связанные с копированием и удалением файлов); Обновленный Toolbox, содержит все макросы в одной панели; Поддержка Python 3; Восстановление рабочего состояния (открытого проекта и файлов) в случае падения редактора. В IDE появился менеджер баз данных и средства публикации проекта (для удобной синхронизации изменений файлов). Я лично пока обхожусь для данных целей Mysql Query Browser и консольной утилитой Unison.
